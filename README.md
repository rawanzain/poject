## How To Make It Work
#### 1- npm install [node_modules] folder 

#### 2- composer install [vendor] folder

#### 3- make .env file

#### 4- php artisan key:generate

#### 5- php artisan migrate:fresh

#### 6- php artisan migrate --seed

#### 7- php artisan serve

#### 8- Enjoy
